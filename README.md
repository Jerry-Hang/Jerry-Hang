# 我的自述

一位16岁的中考落榜生，年轻的垃圾佬，在外人看来与AI聊天比人多的怪人，反感被监控与无隐私，在有目标的时候会死磕即使浪费了很多时间。

为了一个属于自己的电子产品花过1000多块钱买树莓派5，在上面不知死活的刷了win11玩了几天（后来被生态与架构狠狠上了一课），费尽千辛万苦刷安卓14，只为了有一个独属于自己的数字世界。

什么都不懂的情况下玩过洋垃圾，我的第一台电脑就是E5，在Vega矿卡上栽跟头栽过跟头。认识到性能的不足几周不吃中午饭换LGA1700平台.....非常反感用别人的东西为，了一个属于自己的电脑，宁愿自己打工用自己的钱也不想要父母的资助（因为拿人手短，吃人嘴软）。喜欢定制化，只要自己经常用的东西就一定要知其深浅跑在自己的设备上。

本地部署过DeepSeek 1.5B-27B的所有模型，从GTX1060 5G跑4B模型到RTX5060Laptop部署FP4量化14B（虽然说失败了），体验过单路E5平台纯CPU推理27B模型几Token输出的折磨.....不是为了装自己有多厉害，是我害怕在一个我什么都不知道的东西上跟我聊天。

用过vM虚拟机体验从win98到win11的所有win系统，从E5V4锁倍频到V3超频不稳定的突然断电导致BIOS刷坏，为了一张开机3次就烧毁的Vega56矿卡找商家维权.....从本地部署我的世界服务器搞定内网穿透到玩本地AI。

因为身边人兴趣与话题不和天天和AI聊天聊技术（一天8小时，我都没想到聊了这么久），被嘲讽是一个怪人。

现在准备考无线电操作证A证，具体的技术折腾我放在个人[博客](https://jerry-hang.github.io/)这里写不了那么多，消息回的及时度时快时慢，比较懒产能不规律请多多见谅

我也不知道该说什么名人名言了，就随便说一句吧：天将降大任于斯人也，必先苦其心志，劳其筋骨，饿其体肤，空乏其身，行拂乱其所为，所以动心忍性，曾益其所不能
